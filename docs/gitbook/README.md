# GitBook 개발자 센터 — MCP 탭 페이지 구성

## 페이지 구조

```
MCP (탭)
├── MCP 소개              → 01-overview.md
├── 시작하기               → 02-quickstart.md
├── 도구 레퍼런스           → 03-tools.md
└── 활용 가이드             → 04-guides.md
```

## 페이지별 역할

| 페이지 | 목적 | 독자 행동 |
|--------|------|----------|
| MCP 소개 | "이게 뭐고 왜 써야 하지?" 납득 | MCP 개념 이해, 시작하기로 이동 |
| 시작하기 | 5분 안에 연결 완료 | API 토큰 발급 → Claude 설정 → 첫 대화 |
| 도구 레퍼런스 | 어떤 도구가 있고 파라미터가 뭔지 | 필요한 도구 찾아서 활용 |
| 활용 가이드 | 실전 프롬프트와 워크플로우 | 복사해서 바로 사용 |
